Nishan was designed to emulate the actual notion client whether it be web,mobile or desktop as close as possible.
It has a cache layer (under development) adding the following benefits:-

1. Reduce the api call amounts
2. Faster retrieval of data

Nishan has class for almost every type of data there is in notion. eg Block, Page, Space etc, with almost all the required methods.
